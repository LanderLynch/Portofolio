UI/UX overhaul focused on smoother motion, better readability, adaptive scaling, and a cleaner premium presentation layer.

Performance
Unsmooth and laggy animations that could feel heavy during hover and scroll interactions.
Hardware-accelerated smooth UI Motion now leans on transform and opacity for a zero-lag interaction feel.
Viewport Scaling

April 13, 2026

Exact recap of today's portfolio work session.

Today

Homepage Navigation

  • Reworked the homepage navigation a lot:
  • added desktop sidebar / topbar switching
  • changed the sidebar menu order
  • adjusted sidebar vertical position several times
  • added the floating ID / ENG language toggle on the homepage too

Mobile Menu

  • Rebuilt the mobile menu:
  • replaced the older popout with a cleaner glass-style mobile menu
  • removed oversized profile/name blocks from the opened menu
  • reduced mobile button heights
  • fixed the mobile top controls so the chip and hamburger don't sit inside a broken oversized capsule

Zoom And Page Scale

  • Removed zoom UI across the site:
  • hid zoom controls on homepage and shared pages
  • kept page zoom defaulted to 80%
  • removed zoom buttons on mobile only where requested

Language System

  • Added sitewide language switching:
  • created shared language handling in language-controls.js
  • made ID / ENG persist with localStorage
  • translated common labels, nav items, and page text across homepage, certification, and project pages

Guestbook

  • Reworked guestbook features:
  • improved mobile guestbook message layout so avatars can sit beside chat bubbles
  • fixed right-side/admin-author message alignment
  • added custom avatar upload in the avatar chooser
  • stored uploaded avatars in Firebase Storage per user UID
  • kept uploaded avatars reusable for that same signed-in user only

Firebase Storage

  • Updated Firebase Storage setup:
  • defined guestbook-avatars/{userId}/{fileName}
  • added secure upload/delete rules for signed-in owners only
  • limited uploads to image files under 2 MB

Certification Page

  • Improved the certification page:
  • replaced simple click filters with search + type/category filter UI
  • added mobile-friendly stacked filter layout and desktop row layout
  • added softer dropdown-style behavior and smoother filter interactions

Theme Utilities

  • Changed desktop theme/utility placement:
  • moved the palette out of the zoom group and into its own top-left desktop bar
  • kept desktop and mobile behavior separated more clearly

Cursor

  • Added and tuned custom cursors:
  • wired custom cursor support through theme-palettes.css
  • switched to your smaller cursor assets
  • made all cursor states use the same image
  • fixed missing cursor issues caused by oversized source images

Webmeji Integration

  • Integrated Webmeji mascot support:
  • added one roaming mascot across homepage, certification, and project pages
  • fixed drag math for the site's 80% zoom
  • fixed the mascot's drop-to-bottom math
  • renamed the mascot setup from miku to Furina
  • added ChaseMouse and SitAndFaceMouse-style behavior
  • expanded Furina frame usage with v2 / v3 variants where available
  • increased mascot size a few times, currently larger than the original setup

Furina Behavior

  • Reworked Furina behavior closer to Shimeji XML logic:
  • added more literal action-style sequences for sit, look up, spin head, dangle legs, lie down, crawl, run, dash, falling, and climbing
  • added more structured follow-up behavior chains after sitting/chasing/falling

Page Fixes

  • Fixed page-specific issues:
  • fixed the Personal GFX Design Collection page background mismatch by aligning html and body background behavior
  • refreshed multiple cached assets through repeated service worker version bumps so changes actually show up
